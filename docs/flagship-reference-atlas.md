# VisuLive Flagship Reference Atlas

Date: 2026-04-08  
Status: Curated principle board for the current push

This file is the working reference board for the next VisuLive leap.

It is not a moodboard dump.
It is a labeled extraction surface for the team.

## Board Rules

For every reference, record:
- what actually works
- what principle to steal
- what not to steal
- how it maps to VisuLive
- which cue or act it suggests

Use this board for principle extraction, not imitation.
