# Framing Readability Tests

Date: 2026-04-08  
Status: Active reference language

This doc defines the questions VisuLive should use to judge image authority, composition, and distance readability.

The goal is not to make the frame busy.
The goal is to make the frame legible, authored, and different across cue classes.

## Core Rule

If the frame cannot be understood quickly at a distance, it is not yet flagship-strong.

The hero may be important, but the frame must still read when the hero is reduced or displaced.

## Test Distances

### Across the room

Ask:

- what is in charge
- is the frame gathering, spending, or recovering
- is this a wide, narrow, heavy, or open posture
- does the moment feel intentional immediately

### Mid distance

Ask:

- does the chamber read clearly
- do beams, rings, and fields support the cue
- is the image class distinct from neighboring moments

### Up close

Ask:

- do the internal structures reward inspection
- are membranes, cavities, seams, and residues legible
- is detail supporting the image instead of replacing it

## Frame Classes

The system should intentionally produce these classes:

- `Hero Dominant`
- `Chamber Dominant`
- `World Dominant`
- `Split Tension`
- `Collapse`
- `Reveal`
- `Aftermath`
- `Reset`

If the same center-weighted composition dominates every class, the system is still too hero-first.

## Readability Criteria

A frame is strong when:

- the dominant actor is obvious
- the silhouette is clear
- the frame has a readable scale posture
- negative space is used intentionally
- the chamber can take the lead when needed
- the image changes decisively on cue
- aftermath is visible

A frame is weak when:

- it only reads well because the center object is bright
- chamber and hero look interchangeable
- cue changes only affect local motion
- the whole image feels like one repeated pose with different brightness

## Distance Readability Questions

### Hero

- Can the hero read as an emblem, not a blob?
- Does the hero still feel strong when it is smaller?
- Can the hero take most of the screen without erasing structure?

### Chamber

- Can the chamber take over the frame?
- Does the chamber look like architecture, not decoration?
- Can the chamber read as a spatial author in at least some cues?

### Whole Frame

- Does the frame change when the cue matters?
- Is the same image repeating under different cue names?
- Is the viewer seeing stage behavior or object animation?

## Strong / Weak Still Selection

When selecting stills, prefer:

- frames with a clear dominant actor
- frames with a readable cue class
- frames with obvious silhouette and negative-space difference
- frames where aftermath is visible

Reject or down-rank:

- washout frames where structure disappears
- frames that only look good because of brightness
- frames where the hero is the only readable thing
- frames where the chamber is present but not authoritative

## Anti-Goals

Do not mistake these for success:

- more motion
- more glow
- more micro-detail
- more layers
- more hero size
- more bloom

Those are only useful if they strengthen framing, image class, and cue consequence.

## Manual Review Pass

For each candidate frame, answer:

- What is the dominant actor?
- Is the chamber visible as an author?
- Is the frame posture large, small, open, or compressed?
- Does the image class differ from the previous strong frame?
- Would this still read if shown for one second across a room?

If two consecutive strong frames answer these questions the same way, the system still needs more framing diversity.
