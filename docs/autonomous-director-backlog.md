# Autonomous Director Backlog Ledger

Date: 2026-04-29
Status: Active deferred-work ledger

This ledger exists so the old Style/Steer surface does not return by accident and so deferred visual ambitions remain explicit.

## Current Rule

VisuLive is an autonomous playable show engine. Normal-use UI should explain the director, not make the operator tune the show into quality.

Allowed normal surfaces:

- `Start Show`
- route choice: `PC Audio`, `Microphone`, or `Combo`
- read-only `Advanced > Director`
- operational `Advanced > Backstage`

Hidden localhost/dev labs may force scenes, motifs, styles, moments, cue profiles, and asset previews. Lab output is exploratory evidence only.

## Active Coherence Work

- Preserve the read-only Director Console as the normal Advanced surface.
- Serialize `directorConsoleSnapshot` into captures, run journals, samples, and manifests.
- Keep proof-locked route/source/lab/replay/capture mutations behind one app-layer lock path.
- Keep `moment:sheet` review artifacts package-local so archive moves do not break evidence links.
- Continue proving the five playable scenes and four signature moments through no-touch capture, contact sheets, and temporal strips.

- deepen `neon-cathedral` with stronger portal/vault silhouettes and architecture-specific ring posture

## Deferred Hero/World Wave

Blocked until primary coherence proof and operator-trust proof hold:

- hero-suppressed / world-as-hero states

- first owned world mutation verbs
- clearer hero silhouettes that support the active scene rather than competing with it
- world-led frames where the hero is visibly demoted without the show feeling empty

## Deferred Compositor And Memory Wave

Blocked until signature moments and playable scenes are distinct:

- broader compositor ownership beyond the bounded signature-moment slice
- long-form memory traces across scenes
- richer aftermath grammar that recalls prior impacts without becoming permanent treatment
- stronger temporal strips for precharge, strike/open, hold, residue, and clear phases

## Deferred Asset-Pack Work

Allowed only as small, source-owned, provenance-tracked abstract seeds:

- luma masks
- SDF-like glyphs

Do not import stock VJ loops, copied show imagery, large video plates, or opaque model packs. Assets graduate only if they improve thumbnail readability, scene distinction, or aftermath memory.
