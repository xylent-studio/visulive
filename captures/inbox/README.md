# Capture Inbox

Active auto-saved evidence belongs here.

Use this folder for:
- live quick-start batches
- fresh operator validation runs
- the current tuning inbox that `npm run analyze:captures` and `npm run watch:captures` read by default

Do not keep long-term retained captures here.

Move curated evidence into [canonical](C:/dev/GitHub/visulive/captures/canonical) once it has review value.
Move already-learned or exploratory history into [archive](C:/dev/GitHub/visulive/captures/archive).
