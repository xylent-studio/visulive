# VisuLive Preserved Releases

This folder holds machine-readable and human-readable records for intentionally
frozen public VisuLive editions.

Each preserved edition should contain:

- `release-record.json`
- `release-notes.md`
- `SHA256SUMS.txt`
- packaged artifact assets when available
- proof/evidence copies when available

Do not treat this folder as a build cache.
It is a release-history surface.
